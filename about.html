Built for agents that must answer for themselves.

Lians is regulated AI memory infrastructure, built in the open by Ethan Beirne and Dereck Salazar. The thesis: as agents move into finance, healthcare, and legal work, the question stops being “what does the agent know?” and becomes “what did it know when it decided, and can you prove it?”

What exists today

A benchmark you can re-run

Why we built this

A founder note.

Agent memory today is built for chatbots: accumulate everything, rank by similarity, hope the right thing surfaces. That's fine until the answer moves money, touches a patient, or lands in front of a regulator, then the question isn't whether the agent remembered, it's whether it remembered honestly.

Existing stores track one clock; regulated reality runs on two: when a thing was true, and when you learned it. Lians started as that missing primitive and grew into the control plane around it. It's open source because "trust us" is not an acceptable answer in this market. Read the code, run the eval, and hold us to the standard we're selling. · Ethan Beirne, founder
