The risk

Remembering more can mean trusting less.

Generic memory systems absorb facts as they arrive. That leaves agents exposed to stale facts, poisoned instructions, contradictions, overgeneralized preferences, and updates nobody can explain later. In serious environments, memory is not a cache. It is a decision surface.

Developer preview

Improve memory
without guessing.

Generate candidate improvements, inspect them, then commit only the proposals that meet your policy.

from lian_governor import Memory

memory = Memory()

result = memory.add(
  messages=[{"role": "user", "content": "I am building Lian to compete with mem0."}],
  user_id="user_123", auto_commit=False
)

proposals = memory.improve_from_interaction(
  messages=[...], user_id="user_123", mode="propose"
)
memory.commit_proposals([p.id for p in proposals])
recall = memory.recall(query="What is the user building?", user_id="user_123", explain=True)
