Why agent memory breaks in regulated work

Similarity is not
the same as truth.

AI agents need memory — a store of facts they can recall later. Most memory tools retrieve by similarity. But in finance, healthcare, and legal work, facts change: guidance is revised, prescriptions are switched, contracts are amended. Return the most similar fact and the agent acts on a convincing version of the past. Lians returns the fact that was valid when it mattered.
