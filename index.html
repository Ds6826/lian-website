Capabilities

A control plane, not just a store.

Bitemporal recallpoint-in-time

Every fact carries event time and system time. Ask what an agent knew on any past date.

 Tamper-evident audit chainSEC 17a-4 / WORM

Each event enters an append-only SHA-256 hash chain. Verify the record, don't just trust it.

 Crypto-shred erasureGDPR Art. 17

Destroy a per-subject key to provably erase content while the audit trail survives.

Agent-ready

Building with Claude Code or Codex? Copy one prompt and your agent wires Lians into your codebase.

Integrate Lians (regulated AI memory: bitemporal recall, tamper-evident audit, crypto-shred erasure) into this project.

1. Detect the stack and install the SDK:
   - Python:            pip install lians-sdk            (extras: [langchain] [langgraph] [crewai] [mcp] [all])
   - TypeScript / Node:  npm install @lians-ai/lians
   - Go:                go get github.com/Lians-ai/Lians/agentmem/sdk/go@v0.4.0
   - Java (JVM 11+):    ai.lians:lians-sdk:0.4.0 (Maven Central)

2. Configure via environment only (never hardcode secrets): read LIANS_URL and LIANS_API_KEY.
   If they are unset, add both to .env.example and tell me to get a key at https://www.lians.ai/login
   or self-host: git clone https://github.com/ebeirne/Lians2 && docker compose up --build.

3. Wire memory into the agent loop (no agent loop in this project? skip to 4 and integrate at dev time instead):
   - Write durable facts after meaningful exchanges: mem.add(agent_id, content, event_time, metadata).
   - Recall context before responding: mem.recall(agent_id, query), or mem.recall_at(query, as_of=<ISO date>)
     whenever the answer must reflect what was known or valid at a past point in time.
   - Put structured keys in metadata (ticker / subject / metric) so supersession can close stale
     versions automatically instead of letting revisions pile up.

4. If this project is an MCP host, or you are working inside one (Claude Desktop / Claude Code / Cursor /
   VS Code), skip SDK plumbing and register the MCP server: command "uvx", args
   ["--from","lians-sdk[mcp]","lians-mcp"], env: LIANS_URL, LIANS_API_KEY, LIANS_AGENT_ID.
   uvx ships with uv — if missing, install it first: https://docs.astral.sh/uv (winget/brew/curl).

5. Never delete or overwrite memory rows directly: supersession closes stale facts, and
   POST /v1/erase (crypto-shred) is the only compliant erasure path.

6. Finish with a smoke test: add a fact with yesterday's event_time, assert recall() returns it, and
   assert recall_at(as_of=<a date before the write>) returns nothing. No key or server yet? Run it
   serverless: pip install lians-sdk[local], then LocalLiansClient().
