Home / SDKs

One memory API.
Five native SDKs.

Every client speaks the same REST surface — recall, point-in-time recall_at, snapshot, backtest, crypto-shred erasure, audit-chain verify, and the relationship graph. The fastest path is Python local mode: no server, no Docker, no API key.

Full install guide ↗

Install matrix · all SDKs v0.3.0

Pick your language.

Python
pip install lians-sdkfrom lians import LiansClient · local mode: pip install lians-sdk[local] → LocalLiansClient
TypeScript
npm install @lians-ai/liansimport { LiansClient } from "@lians-ai/lians"
Go
go get github.com/Lians-ai/Lians/agentmem/sdk/go@v0.3.0lians.NewClient(url, key)
Java · JVM 11+
dev.lians:lians-sdk:0.3.0Maven coordinates · new LiansClient(opts)
C · C99
cmake -B build && cmake --build buildneeds libcurl · lians_client_new(...)

Honest install state: Python & TypeScript ship from package registries · Go resolves via the module proxy · Java is published as Maven coordinates · C builds from source. All five release in lock-step at the same version.

Self-host the server

Postgres + pgvector,
in your perimeter.

Every SDK except Python local mode talks to a Lians server.

# Postgres + pgvector + API on :8000 git clone https://github.com/Lians-ai/Lians cd Lians docker compose up --build

Health check: curl localhost:8000/livez. See deploy.md for KMS, the non-superuser DB role for RLS, air-gap mode, and WORM storage.

First-class support for LangChain, LangGraph, CrewAI, the OpenAI Agents SDK, and AutoGen v0.4.

MCP — native tool in any AI client

Eight memory tools,
zero adapter code.

Lians is on the official MCP Registry. Claude Desktop, Cursor, VS Code, and Windsurf connect with a one-time config and get eight tools automatically: remember, recall, recall_at, reconstruct, list_conflicts, memory_lineage, fact_history, and backtest_check.

{ "mcpServers": { "lians": { "command": "uvx", "args": ["--from", "lians-sdk[mcp]", "lians-mcp"], "env": { "LIANS_URL": "https://your-lians-server.internal", "LIANS_API_KEY": "lians_...", "LIANS_AGENT_ID": "trading-desk-1" } } } }

Any other MCP host: uvx --from 'lians-sdk[mcp]' lians-mcp with LIANS_URL, LIANS_API_KEY, and optionally LIANS_AGENT_ID in the environment.
